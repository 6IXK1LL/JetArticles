<h1 style="font-weight: bold">
    ⚙️How to activate this bot?
</h1>

### 1. ☔️ **Activate virtual env**:
1. 🐧 **Linux**:
```bash
source ./venv/bin/activate
```
2. 🏠 **Windows**:
```
.\venv\Scripts\activate
```

### 2. 🐍 **Run file** [`bot.py`](https://github.com/6IXK1LL/JetArticles/blob/main/bot.py):
```bash
python3 bot.py
```
